# Contributing to Remindino

Thank you for your interest in contributing to Remindino! We welcome contributions from everyone.

## Getting Started

1.  **Fork the repository** on GitHub.
2.  **Clone your fork** locally:
    ```bash
    git clone https://github.com/yourusername/remindino.git
    cd remindino
    ```
3.  **Create a virtual environment** and install dependencies:
    ```bash
    python3 -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt
    ```
4.  **Create a branch** for your feature or bug fix:
    ```bash
    git checkout -b feature/my-new-feature
    ```

## Making Changes

-   Follow the existing code style.
-   Write clear and concise commit messages.
-   Add tests if applicable.

## Submitting a Pull Request

1.  Push your changes to your fork.
2.  Open a Pull Request on the main repository.
3.  Describe your changes and why they are necessary.

## Code of Conduct

Please be respectful and inclusive in your interactions with the community.

## Reporting Issues

If you find a bug or have a feature request, please open an issue on the issue tracker.
